History of Bash and Zsh

  • Both are based off of the Bourne Shell(sh)
  • Interprets and executes user defined commands
  • Enables the writing and executing of shell scripts

Differences

  • Most commands/scrips can be run successfully on either shell
  • zsh has more complete auto completion
  • zsh has auto correction
  • Plugin support
